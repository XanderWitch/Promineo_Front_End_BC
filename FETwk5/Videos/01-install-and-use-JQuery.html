Installing and Using JQery

JQuery - popular library that makes Javascript simpler. Good for reading from the dom and manipulating or changing values, interacting with nodes, etc.

How to Install JQuery by Itself

Run 'npm init' from command prompt w/in the correct folder. Then run 'npm i jquery'.

Using JQuery

Add as a script to HTML file. ("script src:'node_modules/jquery/dist/jquery.js").

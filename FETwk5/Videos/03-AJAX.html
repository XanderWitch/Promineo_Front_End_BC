AJAX

Sometimes, you only need a section of your page to re-render without updating the whole page.

We can use AJAX, asynchronous JavaScript and XML, to send an http request and only update a section of the DOM. AJAX allows us to send an http request behind the scenes (asynchronously) without having to reload the entire page.

Using AJAX with JQuery

With JQuery's Built-In Methods Get and Post

Both Get and Post request data from a server. However, Post allows us to send a payload to the server to update information on the server.

JQuery returns elements wrapped inside a JQuery object that has built-in methods that allow us to easily interact with that part of the DOM.

JQuery Methods

How to get and set text and values.

JQuery uses a name with a $ symbol to interact with elements of the DOM.

We can also use JQuery to access and changed attributes of elements.

Adding Elements to the DOM with JQuery with Methods

  1. Append: adds context to an existing element at the end of the element's current content
  2. Prepend: adds context to an existing element at the beginning of the element's current content
  3. Before: Add elements and content immediately before the existing element
  4. After: Add elements and content immediately after the existing element

Removing Elements from the DOM with JQuery with Methods

  1. Remove: deletes the element and all its children from the DOM
  2. Empty: removes all the children of the selected element from the DOM

Hide and Show

  1. Hide: Hides an element from being displayed with removing it
  2. Show: Shows an element

Installing Bootstrap

What Is Bootstrap?

Bootstrap is a well-known CSS framework/library that provides an in-depth starting point for many common CSS styles components. Useful for making something use really good, really quickly.

We can load Bootstrap with a CDN (content delivery network), which allows us to link the files over the internet or install them locally and use them in our projects. We'll do the latter.

Installing Bootstrap

From the CLI, go to your project root directory. Run "npm init," enter through everything to choose the default values. This will add a package.json file to your directory and make it an npm project.

Next, use npm to install different libraries. Then on the CLI, type "npm i bootstrap" or "npm install bootstrap." You will now notice a "node modules/bootstrap" directory in your project with several subdirectories.

We also need jquery, so run "npm i jquery" in the directory for the proejct.

Test Out Bootstrap

  1. Link to 
                <link rel="stylesheet" href="node_modules/bootstrap/dist/css/bootstrap.css">
    in the head.
  2. Try 
            <div class="alert alert-primary" role="alert"> It Works!</div>
  3. Link to jquery with 
                <script src="node_modules/jquery/dist/jquery.js"></script> and <script src="node_modules/bootstrap/dist/js/bootstrap.bundle.js"></script>

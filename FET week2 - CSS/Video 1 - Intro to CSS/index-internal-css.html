Internal CSS allows us to put all of our CSS in one place within each HTML document, in a style element in the head. Then we can use selectors to identify which elements the CSS will apply to.

Can select selectors by body, class, or ID.

A CSS class is a selector that allows us to select and name a bit of HTML to add stle to.

An ID selector works similarly to the class selector but identifying it by its ID and are defined by # instead of a dot (>).

If we have multiple pages, we would have to copy internal CSS to all of those pages, and thus creating the same problemes as with inline CSS.

***The text on this page is styled with inline CSS (CSS added directly to the HTML coding).*** Difficult to manage overall. Using stylesheets/INTERNAL CSS is more efficient and less prone to human error.

Internal CSS allows us to put all of our CSS in one place within each HTML document, in a style element in the head. Then we can use selectors to identify which elements the CSS will apply to.

Can select selectors by body, class, or ID.

CSS Values and Units (notes from MDN Web Docs)

A CSS Value (value type = data type) defines the set of values that are allowed for a property in CSS. They are usually distinguished from properties by angle brackets (e.g., color property vs. <color> data type).

Some Numeric Data Types

  1. <integer> - whole number.
  2. <number> - decimal number that may or may not have a decimal point.
  3. <dimension> - a number with a unit attached to it (e.g., 90deg, 10px). This category includes the <length>, <angle>, <time>, and <resolution> types.
  4. <percentage> - a fraction of some other value (e.g., 50%, or an element's link relative to its parent's link).

Lengths

Absolute Length Units

Includes units not relative to anything else; always the same size. Examples:

  • cm
  • mm
  • Q (quarter millimeters)
  • pc (picas)
  • pt (points)
  • px

Relative Length Units

Relative to something else (e.g., size of the parent element's font, size of viewport). Good for scaling and responsive development. Examples:

  • em (font size of the parent or font size of the element itself in properties like width)
  • ex (x-height of the element's font)
  • ch (advance measure [width] of the glyph "0" of the element's font)
  • rem (font size of the root element)
  • lh (line height of the element)
  • vw (1% of the viewport's width)
  • vh (1% of the viewport's height)
  • vmin (1% of the viewport's smaller dimension)
  • vmax (1% of the viewport's larger dimension)

Percentages

Often treated the same way as a length. Always relative to another value. With font size, it will be relative to the font size of the element's parent. For width, it will be a percentage of the width of the parent.

Numbers

Some value types dont need units. For example, opacity takes values from 0 (fully transparent) to 1 (fully opaque).

Color

Common methods of specifying color with good browser support.

Color Keywords

Examples:

  • aqua/cyan (same)
  • fuscshia/magenta
  • blue
  • black
  • lightgrey/lightgray

Hex RGB Values

Use hexadecimal code (has followed by six digits). Examples:

  • #000000 - Black
  • #CC8899 - Purple
  • #ffffff - White

RGB and RGBA Values

RGB represents color with three values: red, green, and blue (e.g., rgb(2, 121, 139) or rgb(197, 93, 161)).

RGBA colors work the same with the addition of a fourth dimension, an alpha channel with gives transparency (from 0 to 1). For example: rgba(2, 121, 139, .3) shows a transparent blue green.

HSL and HSLA Values

Like RGB and RGBA but use Hue (number), Saturation (percentage), Lightness (percentage) in place of red, green and blue. hsla(188, 97%, 28%, .3) gives the same color and transparence as the rgba(2, 121, 139, .3) value in the paragraph above.

Images

This value type is used wherever and image is a valid value, such as an image file, a URL pointing to a URL, or a gradient. Examples:

  • .image {
    • background-image: url(star.png);
  • }
  • .gradient {
    • background-image: linear-gradient(90deg, rgba(119,0,255,1) 39%, rgba(0,212,255,1) 100%);
  • }

Position

The <position> value type represents a set of 2D coordinates that are used to position an item. The first sets the horizontal position, and the second sets the vertical position. If you only give one value, the other will default to "center."

For example: "background-position: right 40px;"" will set an image 40px from the top and to the right of the container.

Strings and Identifiers

Identifiers are keywords that are not quoted because CSS recognizes them (e.g., color keywords like blue.

You can identify strings in CSS by using value types such as <content>. Strings must be quoted in the CSS.

Functions

A function is a section of reusable code that "can be run multiple times to complete a repetitive task with minimal effort." Examples include the "rgb()", "hsl()", "url()", and "calc()" functions. An example in CSS:

.box {width: calc(20% + 100px);}

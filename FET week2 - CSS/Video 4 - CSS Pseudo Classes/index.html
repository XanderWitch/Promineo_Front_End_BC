CSS Pseudo Classes

CSS has something called pseudoclasses, which let us style an element based on its current state, such as if a mouse is hovering over it, if it is a visited link, it has been selected, etc. They are styled the same way, but with a colon and the state. See CSS.

Note, that we can select very specific elements by using multiple selectors. Good for selecting items inside other elements.
